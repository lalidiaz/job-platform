# Job Tracker

A fullstack application built with Node.js, React.js (Redux), and MongoDB to help users manage and track their job applications efficiently.

## About the Project

Job Tracker is a comprehensive application designed to streamline your job search process. Keep track of all your job applications in one place, monitor their status, and manage your job search journey effectively.

## Key Features

### User Authentication

- Register new account
- Login with existing credentials
- Quick access with demo user account
- Secure JWT token authentication


### Job Management

- Create and track job applications
- Update application status (interview, declined, or pending)
- Edit job application details
- Remove job applications
- View comprehensive list of all applications


### Advanced Functionality

- Filter jobs based on multiple criteria
- Pagination for better data organization
- Sort jobs by various parameters
- Real-time updates


### Technical Stack

#### Backend

- Node.js (Express.js)
- MongoDB with Mongoose
- JWT Authentication

#### Frontend

- React.js
- Typescript
- Redux Toolkit for state management
- Responsive design
- Recharts

## Portfolio Note
This application demonstrates the implementation of:

- Full-stack development with MERN stack
- Complex state management with Redux
- RESTful API architecture
- User authentication and authorization
- Database operations with MongoDB
